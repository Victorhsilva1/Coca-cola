Explique o comando git commit e por que é importante criar uma mensagem descritiva.
- O git commit é uma mensagem descritiva que se aparece no github, para assim identificar qual parte você alterou.

Qual a função do comando git push?
- O push serve para enviar o que está no VScode para o repositorio no GitHub.

- É fundamental se dar constantes commits para ter um melhor entendimento de qual parte especifica você alterou.
